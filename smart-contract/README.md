# setup
````
npm init
npm install hardhat

````


# init
````
npx hardhat

````


# build

````
npx hardhat compile
npx hardhat run scripts/deploy.js --network sepolia
npx hardhat verify --network sepolia 0x5fFe862C8a512703575CacCa1d3c31D4A56C7B3b --contract contracts/faucet.sol:Faucet
npx hardhat verify --constructor-args


````


# test

````
npx hardhat test

````
